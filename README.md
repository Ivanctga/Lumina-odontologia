# Lumina Odontologia Premium

## Descrição

Site institucional para a clínica odontológica Lumina Odontologia Premium, localizada em Belo Horizonte. O site apresenta serviços odontológicos, diferenciais da clínica, galeria de imagens e formulário de contato para agendamento de consultas.

## Tecnologias Utilizadas

- **HTML5**: Estrutura da página
- **CSS3**: Estilização e layout responsivo
- **JavaScript**: Interatividade e funcionalidades dinâmicas
- **Font Awesome**: Ícones
- **Google Fonts**: Fontes personalizadas (Poppins, DM Serif Display, Inter)

## Como Visualizar

Para visualizar o site, simplesmente abra o arquivo `index.html` em qualquer navegador web moderno. Não é necessário um servidor local, pois é um site estático.

## Estrutura do Projeto

```
odonto3/
├── index.html                 # Página principal
├── assets/
│   ├── css/
│   │   └── style.css          # Arquivo de estilos CSS
│   ├── js/
│   │   └── script.js          # Arquivo de scripts JavaScript
│   └── img/                   # Diretório de imagens
└── README.md                  # Este arquivo
```

## Funcionalidades

- **Navegação responsiva**: Menu adaptável para desktop e mobile
- **Seções principais**: Hero, Sobre, Serviços, Diferenciais, Galeria, Contato
- **Formulário de contato**: Para agendamento de consultas
- **Botão flutuante do WhatsApp**: Para contato direto
- **Animações de revelação**: Elementos que aparecem ao rolar a página
- **Botão "Voltar ao topo"**: Para navegação rápida
